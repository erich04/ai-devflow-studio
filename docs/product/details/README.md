# DevFlow Studio Product Details

This folder breaks the product details into focused references. Use it before redesigning screens,
changing workflow behavior, or deciding whether a feature belongs in DevFlow Studio.

## Product Promise

DevFlow Studio turns AI-assisted software work into a reviewable delivery flow. A developer can start
from a raw request, run local AI-assisted work, collect evidence, pass human Gates, and sync safe
summaries to the team without uploading raw repository content or provider secrets.

The product is not a chat surface. It is a workflow and evidence system for small engineering teams.
